# Python 依赖

## Python 版本要求

- Python 3.7.16+

## 基础依赖

pandas>=2.0.0
openpyxl>=3.1.0

## Robot Framework

robotframework>=3.2.2,<4.0.0

## RF 执行能力依赖

xmltodict>=0.13.0
websocket-client>=1.0.0

## MCP Server 依赖

如需 TAPD 支持，需安装：
```bash
# 安装 uv（推荐方式）
# macOS
brew install uv

# Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

如需 YAPI 支持，需 Node.js 环境（通过 npx 运行）：
```bash
# YAPI MCP 通过 npx 直接运行，无需额外安装
# 只需配置环境变量 YAPI_BASE_URL 和 YAPI_TOKEN
```

如需 GitLab 支持，需安装：
```bash
npm install -g @modelcontextprotocol/server-gitlab
```

## 自定义库

JLTestLibrary 是项目自定义的 Robot Framework 测试库，包含特定业务关键字。

### 安装方式

JLTestLibrary 以 zip 形式提供，位于项目 `03-scripts/` 目录下：

```bash
# 解压到 Python site-packages 目录
unzip 03-scripts/JLTestLibrary.zip -d $HOME/Library/Python/3.7/site-packages/

# Windows 系统
unzip 03-scripts/JLTestLibrary.zip -d %USERPROFILE%\AppData\Local\Programs\Python\Python37\Lib\site-packages\
```

### 验证安装

```bash
python -c "import JLTestLibrary; print('JLTestLibrary 安装成功')"
```

## 安装命令

```bash
# 安装所有 Python 依赖
pip install -r requirements.txt

# 或分别安装
pip install pandas>=2.0.0
pip install openpyxl>=3.1.0
pip install "robotframework>=3.2.2,<4.0.0"
pip install "xmltodict>=0.13.0"
pip install "websocket-client>=1.0.0"
```

## 注意事项

1. 建议使用虚拟环境（venv 或 conda）
2. Python 版本必须为 3.7.16 或更高
3. JLTestLibrary 需要手动解压安装
4. MCP Server TAPD 需要 uv（通过 uv 安装）
5. MCP Server YAPI 需要 Node.js 环境（通过 npx 运行）
6. MCP Server GitLab 需要 npm
