### 本步骤内容

读取指定的 RF 用例文件，解析以下内容：

**解析项**：
1. **用例名称** - `*** Test Cases ***` 下的用例名
2. **[Documentation]** - 提取三段式描述（预置条件、操作步骤、预期结果）
3. **[Tags]** - 用例标签（优先级、评审状态）
4. **用例步骤** - 实际执行的步骤

---

### ⚠️ Documentation 格式检查（关键）

**正确的 [Documentation] 格式必须包含三段式**：

```robotframework
[Documentation]    【预置条件】<预置条件描述> 【操作步骤】<操作步骤描述> 【预期结果】<预期结果描述>
```

**格式要求**：
- 必须包含 `【预置条件】`、`【操作步骤】`、`【预期结果】` 三个标记
- 所有内容必须在同一行，不要换行
- 使用中文方括号【】标识

**✅ 正确示例**：
```robotframework
A001-001 创建申请单
    [Documentation]    【预置条件】系统正常运行，测试数据已准备 【操作步骤】调用创建申请单接口，提交商户入网申请 【预期结果】申请单创建成功，返回唯一的申请单号
    [Tags]    P0    未评审
    ...
```

**❌ 错误示例**（将导致 TAPD 转换失败）：
```robotframework
A001-001 创建申请单
    [Documentation]    创建商户申请单，返回 merchUniqueId    # ❌ 缺少三段式标记
    [Tags]    P0    未评审
    ...
```

**自动修复指引**：
如果检测到 documentation 格式不正确，请按以下步骤修复：

1. **分析用例内容**：阅读用例的实际执行步骤
2. **提取预置条件**：确定执行用例前需要满足的前提
3. **描述操作步骤**：用文字描述执行了什么操作（不要写接口名）
4. **明确预期结果**：描述用例执行后应该达到的结果
5. **重构 documentation**：按三段式格式重写

**修复示例**：
```robotframework
# 修复前（错误）
[Documentation]    创建商户申请单，返回 merchUniqueId

# 修复后（正确）
[Documentation]    【预置条件】系统正常运行，测试数据已准备 【操作步骤】调用创建申请单接口，提交商户入网申请 【预期结果】申请单创建成功，返回唯一的申请单号
```

### ⚠️ 常见问题处理

**问题 1: "成功处理 0 个测试用例"**
- **原因**: `[Documentation]` 格式不正确，缺少三段式标记
- **解决**: 返回阶段1，检查并修复 documentation 格式

**问题 2: 生成的 Excel 中前置条件/用例步骤/预期结果为空**
- **原因**: Documentation 中的标记格式不正确（如使用了英文方括号 [] 而非中文方括号【】）
- **解决**: 确保使用 `【预置条件】`、`【操作步骤】`、`【预期结果】` 格式

**问题 3: 脚本执行失败，提示文件不存在**
- **原因**: 文件路径错误或文件不存在
- **解决**: 检查 `robot_file` 路径是否正确，使用绝对路径

- 打开 https://www.tapd.cn
- 进入目标项目

**Step 2: 进入测试用例导入页面**
- 点击左侧菜单 "测试" → "测试用例"

**Step 3: 上传 Excel 文件**

- 系统会自动识别列映射关系

**Step 4: 确认列映射**
- 确认以下列映射正确：
  | TAPD 字段 | Excel 列 |
  |----------|----------|
  | 用例目录 | 用例目录 |
  | 用例名称 | 用例名称 |
  | 前置条件 | 前置条件 |
  | 用例步骤 | 用例步骤 |
  | 预期结果 | 预期结果 |
  | 用例等级 | 用例等级 |
  | 创建人 | 创建人 |

**Step 5: 完成导入**
- 点击 "开始导入"
- 等待导入完成
- 检查导入结果，确认用例数量正确

---

### ⚠️ 导入注意事项

1. **用例目录**: 如果 TAPD 中不存在对应的用例目录，系统会自动创建
2. **重复导入**: 相同名称的用例会被视为重复，可以选择覆盖或跳过
3. **特殊字符**: Excel 中的特殊字符可能导致导入失败，如有问题请检查并清理
4. **文件大小**: 单个 Excel 文件建议不超过 5000 条用例，超出请分批导入

---

**导入后用例内容不完整**
