# 非功能性测试能力

## 核心测试类型

### 1. 性能测试
**定义**：评估软件在不同负载条件下的响应时间、吞吐量、资源占用等性能指标，识别系统瓶颈。

**应用方法与场景**：
- 使用 JMeter 模拟高并发用户访问，进行压力测试与负载测试
- 监控 CPU、内存、网络等资源消耗
- 适用于上线前的非功能性验证

**常用工具**：JMeter, LoadRunner, Gatling, k6

**能力水平示例**：
- 初级：能使用 JMeter 创建基本的压力测试脚本
- 中级：能分析性能监控数据定位瓶颈
- 高级：能设计全链路压测方案并推动性能优化落地

### 2. 安全测试
**定义**：检测软件是否存在安全漏洞，如 SQL 注入、XSS 跨站脚本、CSRF 伪造请求等，保障系统与数据安全。

**应用方法与场景**：
- 使用 OWASP ZAP 或 Burp Suite 扫描 Web 应用的安全漏洞
- 验证用户权限控制、数据加密机制
- 适用于金融、电商等对安全性要求高的系统

**常用工具**：OWASP ZAP, Burp Suite

**能力水平示例**：
- 初级：能识别常见安全漏洞类型并复现
- 中级：能使用工具进行自动化扫描并分析报告
- 高级：能设计纵深防御策略并主导安全攻防演练

### 3. 兼容性测试
**定义**：验证软件在不同操作系统、浏览器、设备型号等环境下的功能表现一致性。

**应用方法与场景**：
- 针对项目测试 Chrome、Firefox、Edge 等主流浏览器的显示与交互效果
- 针对测试不同品牌手机（如华为、小米、iPhone）的功能适配
- 适用于多终端发布的软件产品

**常用工具**：BrowserStack, Sauce Labs

**能力水平示例**：
- 初级：能在指定设备上执行兼容性测试用例
- 中级：能总结跨平台差异报告
- 高级：能搭建自动化兼容性测试矩阵并集成云测试平台

### 4. 可访问性测试
**定义**：确保产品能够为所有用户群体服务，包括残障人士和特殊需求用户，验证视觉、听觉、运动和认知可访问性。

**应用方法与场景**：
- 验证产品是否符合 WCAG 标准
- 测试屏幕阅读器兼容性、键盘导航和颜色对比度
- 适用于所有产品发布前的合规性检查

**常用工具**：axe, WAVE, Accessibility Scanner

**能力水平示例**：
- 初级：能执行基础可访问性检查
- 中级：能识别常见可访问性问题
- 高级：能制定全面的可访问性测试策略

### 5. 可靠性测试
**定义**：一种系统化的方法，用于评估产品在预期使用条件下长期保持功能和性能的能力，包括识别潜在故障模式、预测产品使用寿命、优化设计。

**应用方法与场景**：
- 通过加速寿命测试、环境应力筛选、失效模式与效应分析等方法
- 评估产品在各种环境下的可靠性特征量
- 暴露设计、原材料和工艺流程中的潜在问题

**常用工具**：IEC 60068系列, GB/T 2423系列, MIL-STD-810G/H

**能力水平示例**：
- 初级：能执行基本的环境应力筛选测试
- 中级：能设计完整的可靠性测试方案
- 高级：能进行失效模式与效应分析并优化产品设计

### 6. 恢复测试
**定义**：评估计算机系统或软件在出现故障后能够快速恢复正常运行的能力的方法，通过人为制造系统故障来模拟真实情况下的错误。

**应用方法与场景**：
- 通过强制系统失败验证恢复能力
- 采用各种办法强迫系统出错，使其不能正常工作
- 检验系统的恢复能力

**常用工具**：Chaos Monkey, Gremlin, Datadog, Nagios

**能力水平示例**：
- 初级：能执行基本的恢复测试用例
- 中级：能设计多种故障场景的恢复测试
- 高级：能构建自动化恢复测试框架并分析恢复指标

### 7. 渗透测试
**定义**：模拟黑客攻击手法，对网络系统、应用程序及数据进行安全测试，深入挖掘安全隐患，验证防护机制有效性。

**应用方法与场景**：
- 执行渗透测试、生存性测试等核心任务
- 模拟真实攻击环境，对系统在遭受持续攻击时的稳定性、可用性及数据安全性进行有效评估

**常用工具**：Burp Suite, OWASP ZAP, SQLMap, Metasploit Framework

**能力水平示例**：
- 初级：能使用工具进行基础扫描
- 中级：能独立开展渗透测试工作
- 高级：能设计复杂攻击场景，深入分析漏洞利用路径

### 8. Web 安全测试
**定义**：针对 Web 应用进行安全测试，包括 SQL 注入、XSS 跨站脚本、CSRF 跨站请求伪造、文件上传/文件包含、越权访问等常见漏洞的测试。

**应用方法与场景**：
- 在 Web 应用渗透测试中，使用 Burp Suite 等工具进行抓包、改包、爆破测试
- 验证 Web 应用的安全性

**常用工具**：Burp Suite, OWASP ZAP, sqlmap, Dirsearch

**能力水平示例**：
- 初级：能使用工具扫描基础漏洞
- 中级：能手工验证漏洞真实性
- 高级：能挖掘复杂的业务逻辑漏洞
