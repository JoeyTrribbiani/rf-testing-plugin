# 软件测试能力

本技能是测试能力的主入口，根据用户问题类型按需加载子技能。

## 子技能导航

| 问题类型 | 子技能 | 触发关键词 |
|---------|-------|-----------|
| 测试基础、测试流程、测试设计方法 | `testing_basics` | 需求分析、测试用例、等价类、边界值、场景法、探索性测试 |
| 功能测试、API 测试、单元测试、集成测试 | `testing_functional` | 功能测试、API测试、接口测试、单元测试、集成测试、回归测试 |
| 性能测试、安全测试、兼容性测试 | `testing_nonfunctional` | 性能测试、安全测试、渗透测试、兼容性、可访问性、可靠性 |
| 自动化测试、CI/CD、质量门禁 | `testing_automation` | 自动化测试、UI自动化、CI/CD、质量门禁、测试左移 |
| AI 赋能测试、智能测试 | `testing_ai` | AI测试用例生成、智能测试、AI报告、测试数据生成、缺陷智能分析 |

## Robot Framework 自动化

对于 Robot Framework 自动化测试相关问题，主入口会指引加载 `rf_*` 系列技能：
- `rf_basics` - RF 基础规范与用例结构
- `rf_keywords` - RF 自定义关键字编写
- `rf_jsonpath` - RF JSONPath 表达式与数据提取
- `rf_variables` - RF 变量管理与作用域
- `rf_validation` - RF 响应验证与断言
- `rf_builtin` - RF 内置关键字用法

## 快速决策指南

**根据用户问题快速定位所需子技能**：

1. 用户问"如何设计测试用例"、"如何进行需求分析"、"等价类/边界值怎么用"
   → 加载 `testing_basics`

2. 用户问"如何测试 API"、"如何进行单元测试"、"集成测试怎么做"
   → 加载 `testing_functional`

3. 用户问"性能测试怎么做"、"如何进行安全测试"、"兼容性测试方法"
   → 加载 `testing_nonfunctional`

4. 用户问"自动化测试框架"、"CI/CD集成"、"质量门禁"、"测试左移"
   → 加载 `testing_automation`

5. 用户问"AI 如何生成测试用例"、"智能测试"、"测试报告自动生成"
   → 加载 `testing_ai`

6. 用户问"Robot Framework 如何编写用例"、"RF 关键字如何定义"
   → 加载 `rf_basics`、`rf_keywords` 等对应技能
