### 本步骤内容

调用转换脚本生成 TAPD Excel 文件：

**调用命令**：
```bash
python 03-scripts/robot2tapd.py ${robot_file} \
    --excel ${output_excel} \
    --creator ${creator_name} \
    --out-b64 ${base64_file}
```

**转换参数**：
- `robot_file`: RF 用例文件路径
- `output_excel`: 输出的 Excel 文件名
- `creator`: 创建人名称
- `base64_file`: base64 输出文件路径

**输出**：
- TAPD Excel 文件（包含所有用例）
- 用例数量统计
- 转换耗时
